# Client Service (User Importer)

Микросервис на Spring Boot для импорта пользователей из CSV-файла с валидацией и статистикой обработки.

## Возможности

- Загрузка CSV-файла, сохранение на файловую систему и в БД
- Асинхронная построчная обработка с валидацией полей
- Сохранение валидных пользователей (обновление по телефону или добавление)
- Сохранение ошибочных строк с кодом и описанием ошибки
- Статистика обработки по каждому файлу
- Поиск пользователей с фильтрами и пагинацией

## Стек

- Java 17, Spring Boot 3
- Spring Data JPA, PostgreSQL
- MapStruct, Lombok
- Gradle, Docker Compose

## Эндпоинты

| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/files` | Загрузка файла (201, 409 при дубликате) |
| POST | `/files/{id}/processing` | Запуск асинхронной обработки (204, 404) |
| GET | `/files/statistics` | Список файлов со статистикой (фильтр по статусу) |
| GET | `/files/{id}/statistics` | Детальная статистика с ошибками |
| GET | `/clients` | Список клиентов (фильтры + limit/offset) |

## Формат CSV

`Фамилия,Имя,Отчество,Email,Телефон,ДатаРождения`

```
Иванов,Иван,Иванович,ivan.ivanov@shift.ru,79161234567,1990-05-15
```

Правила: имя/фамилия/отчество — кириллица с заглавной, 3–50 символов (отчество необязательно); email — домены `shift.com`/`shift.ru`; телефон — 11 цифр, начинается с `7`; дата рождения — `yyyy-MM-dd`, возраст ≥ 18.

## Запуск

```bash
docker compose up -d
./gradlew bootRun
```

Приложение поднимется на `http://localhost:8080`, PostgreSQL — на `5432`.
